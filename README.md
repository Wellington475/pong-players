# Pong-players

____

Pong-players is a simple personal project of a mutiplayer game using node javascript technologies (expressjs + socket.io + html5).

___

### Technologies (Node.js)

- [Express](https://expressjs.com/)
- [Socket.io](https://socket.io/)
